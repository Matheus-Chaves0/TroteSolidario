# 🎓 Trote Solidário

Projeto acadêmico desenvolvido com foco em organização, colaboração e experiência web moderna.

O sistema foi criado para auxiliar no gerenciamento e divulgação do **Trote Solidário**, promovendo integração, participação e organização de atividades acadêmicas.

---

## 🌐 Acesse o projeto

🔗 **Deploy Online:**  
https://matheus-chaves0.github.io/TroteSolidario/

---

## 📌 Sobre o projeto

O **Trote Solidário** é uma aplicação web desenvolvida como projeto acadêmico, com o objetivo de praticar conceitos de:

- Desenvolvimento Web
- Estruturação Frontend
- Organização de Layouts
- Responsividade
- Experiência do Usuário (UI/UX)
- Boas práticas de desenvolvimento

---

## 🚀 Funcionalidades

- 📋 Exibição de informações do evento
- 🎨 Interface moderna e responsiva
- 📱 Compatível com dispositivos móveis
- ⚡ Navegação simples e intuitiva
- 🧩 Estrutura organizada de componentes
- 🌐 Aplicação web de fácil utilização

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Matheus-Chaves0/TroteSolidario.git
```

Acesse a pasta:

```bash
cd TroteSolidario
```

Abra o arquivo `index.html` no navegador.
